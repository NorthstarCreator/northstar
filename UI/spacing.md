# Spacing

Northstar should feel spacious but efficient.

## Scale

- Small gap: 8px
- Standard gap: 14-18px
- Section gap: 22-28px
- Page padding: 28-34px desktop
- Mobile page padding: 14-20px

## Rules

- Avoid stacked sections that feel crowded.
- Keep above-the-fold content actionable.
- Prefer 2-column desktop layouts and 1-column mobile layouts.
