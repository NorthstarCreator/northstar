# Charts

Charts are secondary in Northstar. Decisions come first.

## Rules

- Use charts only when they clarify a decision.
- Keep cards compact.
- Prefer insight cards, rankings, and action cards over additional graphs.

## Sizing

```css
.chart-wrapper {
  position: relative;
  height: 240px;
  width: 100%;
}
```

Chart.js should use:

```js
responsive: true,
maintainAspectRatio: false
```
