# Buttons

Buttons should feel like premium software controls.

## Primary

- White or dark ink background depending on context
- 14-16px radius
- Subtle shadow
- Clear verb label

## Secondary

- White background
- Minimal border
- Hover lift of 1-2px

## Danger

- Muted red text
- Use only for reset, retire, or destructive actions

Buttons should not stretch full width unless used in mobile or card layouts.
