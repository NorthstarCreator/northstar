Privacy Policy

Last updated: July 5, 2026

NorthStar Creator is designed as a creator operating system. The current local version stores creator business data on the user's own device unless the user intentionally exports, imports, syncs, or connects an external service.

Information Northstar May Store Locally

NorthStar may store creator data such as products, videos, sample records, sales records, notes, hooks, generated plans, settings, and backup metadata in local browser storage, bundled local files, or approved connected systems.

How Data Is Used

Data is used only to provide the product experience, including dashboards, daily briefs, planning tools, and performance insights. User data is not sold.

Third-Party Connections

When TikTok authentication is enabled, OAuth access tokens will be handled through a secure server-side connector using industry-standard practices. Access tokens and refresh tokens will not be stored in browser localStorage. NorthStar will only access TikTok data after a user explicitly authorizes the connection through TikTok's official OAuth flow.

Backups and Exports

Users may export backups from Northstar. Exported files are controlled by the user and should be stored carefully.

Contact

Users may request deletion or contact support through the Contact page or by emailing hello@northstar-creator.com.
