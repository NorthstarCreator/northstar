# Northstar Live Dashboard Sandbox

This folder is a visual-only prototype for the future hosted Northstar dashboard.

## Safety

- Mock data only.
- No TikTok API calls.
- No OAuth code.
- No credentials.
- No production dashboard files imported.
- No backend required.

## Open Locally

Double-click:

`dashboard/sandbox/dashboard/index.html`

The prototype works directly from `file://`.

## Included Screens

- Morning Brief
- Opportunity Center
- Earnings
- Products
- Product Workspace
- Videos
- Video Detail
- Data Hub
- Settings

## Demonstrated Interactions

- Sidebar navigation
- Account switching
- Date filtering
- Custom date range
- Earnings-source filtering
- Product detail navigation
- Video detail navigation
- Product video sorting
- Mock content generation
- Add Account mock modal
- Add Revenue Source mock modal

## Notes

This prototype is intentionally calm, compact, and non-technical. It is designed for visual review before any future sandbox TikTok integration work.
