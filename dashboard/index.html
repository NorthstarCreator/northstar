Data Safety Notice

The Vercel-hosted dashboard shell may be publicly reachable. Personal creator data is available only within the authenticated user session/browser and is never bundled into deployed static files.

  • No TikTok client secret is stored in this dashboard.
  • No TikTok access token or refresh token is returned to browser JavaScript.
  • No Jennifer local records, exports, backups, notes, sales records, videos, or samples are deployed with this shell.
