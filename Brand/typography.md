# Northstar Typography

Northstar should feel premium, direct, and quiet.

## Font Stack

Use the system UI stack:

```css
font-family: Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
```

## Scale

- App title: 48-96px
- Page title: 30-52px
- Section title: 20-24px
- Card title: 16-20px
- Body: 14-16px
- Metadata: 12-13px

## Rules

- Letter spacing should stay at `0` except small uppercase labels.
- Avoid oversized text inside cards.
- Use bold weight for decisions, not decoration.
