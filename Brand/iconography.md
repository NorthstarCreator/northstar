# Northstar Iconography

Icons should be minimal, familiar, and functional.

## Style

- Rounded corners
- Simple geometric forms
- No heavy illustration
- Use icons to clarify navigation and status

## Workspace Symbols

- Raised Right: heart, warm red/gold
- Truth Tuned Tribe: blue heart, teal/blue
- Northstar: star/compass mark

## Status Symbols

- Success: check
- Watch: clock or eye
- Wait: pause
- Retire: muted warning

Avoid decorative icon clutter. Every icon should help scanning.
